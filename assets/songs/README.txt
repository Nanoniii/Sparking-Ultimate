=== DBSU — Músicas ===

Coloque os arquivos de música nesta pasta com os seguintes nomes:

1. menu_main.mp3       → Tema do Menu Principal
2. menu_charselect.mp3 → Tema da Seleção de Personagens
3. battle_normal.mp3   → Música de batalha padrão (Versus / Z-Canon)
4. battle_boss.mp3     → Música para batalhas de chefe (ch7 Vegeta, etc.)
5. story_intro.mp3     → Tema de abertura / Z-Canon menu
6. victory.mp3         → Tela de vitória / resultado
7. training.mp3        → Modo Treino
8. tension.mp3         → Tema de tensão (diálogos dramáticos)

Formato recomendado: MP3 ou OGG
